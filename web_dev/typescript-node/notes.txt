1) `npm init -y`

2) `npm i -D typescript ts-node nodemon`
  /* 'ts-node' is required to automatically update the changes */

3) `npx tsc --init`
  /* this will create tsconfig.json file */

***
first three lines are important and they are need to get executed in order
and now rest commands, as you required
***

4) `npm i -D @types/node @types/express`
5) `npm i express`
6) `npm i -D eslint @typescript-eslint/eslint-plugin @typescript-eslint/parser`
7) `npm i -D prettier eslint-config-prettier eslint-plugin-prettier`
