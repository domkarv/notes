1) `npm init -y`

2) `npm i -D typescript ts-node nodemon`
  /* 'ts-node' is required to automatically update the changes */

3) `npx tsc --init`
  /* this will create tsconfig.json file */
  /* now add the following lines in tsconfig.json file */
  {
    "compilerOptions": {
      "target": "es2016",
      "module": "commonjs",
      "esModuleInterop": true,
      "forceConsistentCasingInFileNames": true,
      "strict": true,
      "skipLibCheck": true,
      "moduleResolution": "Node"
    }
  }
  if above json snippet is present as it is then it's ok, else is there anything different, then cross check it

***
first three lines are important and they are need to get executed in order
and now rest commands, as you required
***

5) `npm i express`
